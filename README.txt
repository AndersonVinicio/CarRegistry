Sprint 3.C. Java (Mar24) - Lab 8 : La gestión de archivos y recursos
   En este lab se trabajar con ficheros para recibir,
   enviar o almacenar información. Spring Boot y Java nos facilitan estas tareas
   gracias a las herramientas de manejo de recursos y ficheros. En el fastbook que te
   proponemos a continuación, aprenderás cómo implementar estas tareas dentro de las
   aplicaciones y cómo recibir y enviar archivos a los usuarios por medio de las APIs.
